This routine is used if aggregation is enabled, data is already on the io-tasks

Parameters
[in]file,:a pointer to the open file descriptor for the file that will be written to
[in]nvars,:the number of variables to be written with this decomposition
[in]vid,:an array of the variable ids to be written
[in]iodesc_ndims,:the number of dimensions explicitly in the iodesc
[in]basetype: the basic type of the minimal data unit
[in]gsize: array of the global dimensions of the field to be written
[in]maxregions: max number of blocks to be written from this iotask
[in]firstregion: pointer to the first element of a linked list of region descriptions.
[in]llen: length of the iobuffer on this task for a single field
[in]maxiobuflen: maximum llen participating
[in]num_aiotasks: actual number of iotasks participating
[in]IOBUF,:the buffer to be written from this mpi task
[in]frame: the frame or record dimension for each of the nvars variables in IOBUF
